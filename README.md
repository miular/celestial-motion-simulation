# celestical-motion-simulation
使用python语言编写的天体运动模拟

---
##使用方法
1.根据galaxy.json文件有的星系，选择感兴趣的填入这一行
‘’‘
SYSTEM_CONFIG = load_system('galaxy.json','你选择的星系')
’‘’
2.也可以自创一个星系直接写入你的galaxy.json文件里面
